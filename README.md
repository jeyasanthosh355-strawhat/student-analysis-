# 🎓 Student Performance Analysis System

## 📌 Overview

The **Student Performance Analysis System** is a data science project that uses Python to generate and analyze student academic data. The project focuses on performing **Exploratory Data Analysis (EDA)** to identify patterns, trends, and insights in student performance.

This system helps in understanding how factors like marks, attendance, and internal scores affect overall performance.

---

## 🎯 Objectives

* Generate a synthetic dataset of student records
* Perform data cleaning and preprocessing
* Analyze student performance using statistical methods
* Identify top-performing and low-performing students
* Compare performance based on department, gender, and year
* Study relationships like attendance vs marks
* Visualize data using graphs and charts

---

## 🧾 Dataset Details

The dataset contains **250 student records** with the following attributes:

* Student_ID
* Name
* Gender
* Department (CSE, IT, ECE)
* Year (1st, 2nd, 3rd)
* Maths Marks
* Science Marks
* English Marks
* Attendance (%)
* Internal Marks
